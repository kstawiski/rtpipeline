Key Features

Everything you need for radiotherapy research data processing

AI-Powered Segmentation

Automatic organ segmentation using TotalSegmentator with 100+ anatomical structures. Support for custom nnU-Net models.

DVH Analysis

Comprehensive dose-volume histogram calculation with 50+ clinical metrics including Dmean, Dmax, Vx, Dx, and conformity indices.

Radiomics Extraction

IBSI-compliant feature extraction with PyRadiomics. Shape, first-order, texture, and filtered features from CT/MR images.

Batch Processing

Snakemake workflow orchestration for processing hundreds of patients. Automatic parallelization and fault tolerance.

Docker & Singularity

Pre-built containers with all dependencies. Run on local workstations, HPC clusters, or cloud platforms.

Quick Start

Get up and running in minutes

Recommended

Docker

Pre-built container with all dependencies and model weights.

# Pull the image
docker pull kstawiski/rtpipeline:latest

# Run the pipeline
docker run --gpus all \
  -v /path/to/dicom:/data/input \
  -v /path/to/output:/data/output \
  kstawiski/rtpipeline:latest \
  snakemake --cores all

Installation

Docker Installation

1

Prerequisites

  • Docker 20.10+
  • NVIDIA Container Toolkit (for GPU)
  • At least 32GB disk space
2

Pull the Image

docker pull kstawiski/rtpipeline:latest
3

Run the Pipeline

docker run --gpus all --rm \
  -v /path/to/dicom:/data/input:ro \
  -v /path/to/output:/data/output:rw \
  -v /path/to/logs:/data/logs:rw \
  kstawiski/rtpipeline:latest \
  snakemake --cores all --configfile /app/config.container.yaml

Conda Installation

1

Clone Repository

git clone https://github.com/kstawiski/rtpipeline.git
cd rtpipeline
2

Create Environments

mamba env create -f envs/rtpipeline.yaml
mamba env create -f envs/rtpipeline-radiomics.yaml
3

Configure and Run

cp config.yaml.example config.yaml
# Edit config.yaml with your paths

conda activate rtpipeline
snakemake --cores all --use-conda

Singularity Installation

1

Convert Docker Image

singularity pull rtpipeline.sif docker://kstawiski/rtpipeline:latest
2

Run with GPU

singularity exec --nv \
  --bind /path/to/dicom:/data/input:ro \
  --bind /path/to/output:/data/output:rw \
  rtpipeline.sif \
  snakemake --cores all --configfile /app/config.container.yaml
